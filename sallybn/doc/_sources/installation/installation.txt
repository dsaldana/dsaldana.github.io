*******************************
Installation
*******************************


* `Debian and Ubuntu`_
* `Linux`_
* `Windows XP, 7, 8`_


Debian and Ubuntu
=================
To install in Debian or Ubuntu:

1. Go to the `download site <http://dsaldana.github.io/sallybn/download.html>`_.

2. Download the *Debian version*.

3. Run the command::

    $ sudo deb -i sallybn_*.deb

4. Execute Sally-BN::

    $ sallybn


Linux
=======
Any linux distribution:

1. Install `python 2.7 <https://www.python.org/downloads/>`_.

2. Install `Python GTK+ 3 <http://python-gtk-3-tutorial.readthedocs.org/en/latest/install.html>`_.

3. Go to the `download site <http://dsaldana.github.io/sallybn/download.html>`_.

4. Download the *Linux package version*.

5. Extract the package

    $ tar -xvf sally*.tar.gz

6. Enter to the folder::

    $ cd sally*

7. Execute Sally-BN::

    $ ./sallybn


Windows XP, 7, 8
================

1. Go to the `download site <http://dsaldana.github.io/sallybn/download.html>`_.

2. Download the *windows version*.

3. Extract the zip file.

4. Double click on the file **SALLYBN.bat**
